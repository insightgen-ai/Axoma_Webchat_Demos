# AXOMA Webchat Dashboard

## 🚀 Overview

AXOMA Webchat Dashboard is a modern, responsive web application that provides specialized AI-powered chatbot interfaces for four industrial domains:

- **Electric Powergrid Webchat** - Real-time grid telemetry, load forecasting, and fault detection
- **Axoma-Airway Webchat** - Flight coordination, weather-aware ATC assistance
- **Axoma-Banking Webchat** - Fraud detection, compliance monitoring, financial intelligence
- **Axoma-Electricals Webchat** - Smart device diagnostics, motor protection, IoT asset management

Built with Bootstrap 5 and Font Awesome, this dashboard offers a unified interface to access domain-specific AI agents.

## 📋 Prerequisites

- **Docker** (version 20.10.0 or later)
- **Git** (optional, for cloning)
